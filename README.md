# Mini Music Generator

Простой проект на React + TypeScript + Tailwind CSS + Tone.js.

## Быстрый старт

1. Установите зависимости:

```bash
npm install
```

2. Запустите проект:

```bash
npm run dev
```

3. Откройте браузер по адресу, который покажет Vite.

## Что здесь

- React + TypeScript
- Tailwind CSS
- Tone.js для воспроизведения мелодии

## Упражнение

Измените `src/App.tsx`, чтобы создать свою мелодию, ритм или синтезатор.
